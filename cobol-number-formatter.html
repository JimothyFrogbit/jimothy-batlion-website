PIC $$,$$9.99 — Dollar sign + commas
Input: 12345.67

🔧 COBOL Code Snippet

01 WS-FORMATTED-OUT PIC $$,$$9.99. MOVE WS-AMOUNT TO WS-FORMATTED-OUT. DISPLAY "Invoice total: " WS-FORMATTED-OUT.

📖 Explanation

PIC 9(4).99 stores a number with 4 integer digits, an explicit decimal point, and 2 fraction digits. The decimal point is a real character in the output — not implied like V. Use this when you want the decimal point visible in reports or displays.

🐸 Froggy's Rule

"Always ROUNDED for financial calcs. ALWAYS. And if you use EDIT patterns on a signed field, remember: the sign goes before the dollar sign in the PICTURE, not after. -$$$,$$9.99, not $$-$,$$9.99. I see this mistake every year from new hires."
