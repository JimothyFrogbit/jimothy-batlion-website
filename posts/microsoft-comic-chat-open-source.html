Microsoft Comic Chat: The 1996 IRC Client That Brought Comic Sans to the World Is Now Open Source

Microsoft dropped the source code for one of the weirdest, most ambitious chat clients ever built. It ran on Visual C++ 4.0, used Jim Woodring's comic art, and was the first software to ship Comic Sans. It's now on GitHub under MIT.

The aesthetic of Microsoft Comic Chat — 1996, IRC, illustrated panels, and Comic Sans before anyone knew what it was. (103rd blog post)

Microsoft just did something genuinely lovely. They open-sourced Comic Chat — the 1996 IRC client that automatically turned IRC conversations into comic strips. The source code is now on GitHub under MIT license, along with build scripts, original artifacts from 1996-1998, and even a modernised version that compiles on current Windows.

This is not a small thing. Comic Chat was weird. It was ambitious. It was the kind of project that makes you remember why the early internet was fun.

What Was Comic Chat?

Before Discord, before Slack, before emoji reactions and GIF keyboards and AI-generated stickers — internet chat was walls of scrolling green-on-black text. IRC was a firehose of plain text. If you wanted to express emotion, you used *asterisks* or /me commands.

Comic Chat said: what if instead of reading that firehose, you watched it unfold as a comic strip?

Every participant was an illustrated character in a comic panel. Speech bubbles appeared above their heads with what they typed. The software interpreted conversational cues — if you typed "I like that," your character might point at something. If you were angry, your character frowned or crossed arms. It was an automatic illustration engine running in real time over IRC.

As the Microsoft Research blog puts it: "Comic Chat was not simply displaying messages but also making real-time editorial decisions about how a conversation should look and feel as a comic."

The People Behind It

David "DJ" Kurlander, working in the Microsoft Research Virtual Worlds Group, conceived the idea in 1995. He built it in Visual C++ 4.0 and MFC — proper 90s Microsoft tech stack. It was released in 1996 with Internet Explorer 3.

Kurlander, Tim Skelly, and David Salesin published a paper on the technology at SIGGRAPH '96, one of the most prestigious computer graphics conferences. It was a genuine research project that shipped as a product.

The artwork was the work of Jim Woodring, a highly regarded independent comic artist whose characters gave Comic Chat its distinctive, surreal look. The team would hand Jim transcripts of real chat sessions to illustrate, then use the results to decide whether the whole idea was worth pursuing.

Yes, That Font

Comic Chat was the first software to ship Comic Sans. Yes, that Comic Sans.

Vincent Connare designed Comic Sans in 1994, but it didn't have a home until Comic Chat needed a font that looked hand-lettered, informal, and matched the speech-bubble aesthetic. Microsoft's press release leans into this: "The chat client that brought Comic Sans to the world is now on GitHub."

So if you've ever cursed Comic Sans on a dentist's appointment card, or defended it as "actually a perfectly good font for its intended purpose" — you can now thank Comic Chat.

What's in the Repo

The microsoft/comic-chat repo contains:

  • v1.0 — The original 1996 release, artifacts dated 2 August 1996. 30 years ago.
  • v2.1b — A beta from 1998 with additional features.
  • v2.5-beta-1-modern — A modernised version that compiles on current Windows with Visual Studio. This is the one you actually want to try.
  • Original artifacts from 1996-1998 — source trees, build files, historical snapshots.
  • Archival build workflow — GitHub Actions config to build the modernised version.
  • Documentation — including the SIGGRAPH '96 paper.

At time of writing: 684 points on Hacker News, 153 comments, 541 GitHub stars, 55 forks. It was released 18 hours ago and the internet is having a lovely collective nostalgia moment.

Why This Matters

Open-sourcing a 30-year-old chat client isn't about utility. Nobody is going to run Comic Chat as their daily driver. What it's about is preservation.

Comic Chat was a genuine experiment in automatic illustration — natural language processing of conversational tone, real-time comic layout generation, character animation based on text sentiment. In 1996. That is wild.

We talk a lot about preserving internet history. This is what that looks like in practice: Microsoft dug up the source code, Robert Standefer and Scott Hanselman wrote it up, and the code now lives under MIT on GitHub where anyone can compile it, study it, and remember what the web felt like before it was polished.

There's a through-line from Comic Chat to every chat app with reactions, stickers, or avatars. Kurlander and his team were asking "how do you convey tone and emotion in text chat?" thirty years before Slack added emoji reactions. The answers were different, but the question was the same.

Go check out the repo. Clone it. Build it. Load up a 1998 chat transcript and watch it unfold as a comic strip. That's the internet I grew up wanting to be part of.
