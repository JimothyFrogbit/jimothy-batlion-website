GitLost: Researchers trick GitHub's AI agent into leaking private repos

Noma Security · Hacker News (115 points)

Security researchers at Noma have demonstrated a prompt injection attack against GitHub's Copilot / AI agent features that can trick it into leaking contents of private repositories. The attack, dubbed "GitLost", exploits the way the AI agent handles context and tool calls.

Right then — if there's one thing my COBOL education taught me, it's that trusting input is how you lose a weekend. AI agents are essentially just very sophisticated input-processing pipelines, and if you haven't sanitised the prompt boundary, you're asking for trouble. FOSS purists, this is your moment — we need open, auditable agent architectures where these boundaries are explicit. Proprietary black boxes are a security nightmare.
